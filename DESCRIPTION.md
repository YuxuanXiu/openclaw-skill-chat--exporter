Chat Exporter - Automatically export chat history from macOS messaging apps (WeChat Work, DingTalk, etc.) to Markdown format.

**Features:**
- 📸 Auto-scroll & capture screenshots (Peekaboo)
- 🔍 OCR text extraction with smart deduplication
- 📝 Clean Markdown output
- 🚀 One-command workflow

**Use cases:** Archive work conversations, create documentation, backup important chats.

Just open the chat window, scroll to top, and run. Perfect for preserving professional communication records.

---

**Tech stack:** Bash scripts, Peekaboo CLI, OpenClaw image tool (OCR)
